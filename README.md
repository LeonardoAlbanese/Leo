# Leo Side Scroll

Juego de scroll lateral en Electron. Recoge monedas para subir tu puntaje y esquiva enemigos simples en el camino.

## Requisitos

- Node.js 18+.

## Instalar dependencias

```bash
npm install
```

## Ejecutar en modo escritorio

```bash
npm start
```

## Generar ejecutable

```bash
npm run dist
```

> El resultado queda en la carpeta `dist/` para tu sistema operativo.
